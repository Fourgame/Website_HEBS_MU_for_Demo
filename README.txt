นี่คือร่างไฟล์ README.md สำหรับโปรเจกต์เว็บไซต์ Health Education Mahidol University ของคุณ โดยปรับปรุงเนื้อหาให้ครอบคลุมส่วนประกอบต่างๆ ที่ปรากฏในเว็บไซต์เดโมครับ

Website Health Education Mahidol University (Demo)
โปรเจกต์นี้เป็นเว็บไซต์ต้นแบบ (Demo) สำหรับ ภาควิชาสุขศึกษาและพฤติกรรมศาสตร์ คณะสาธารณสุขศาสตร์ มหาวิทยาลัยมหิดล เพื่อนำเสนอข้อมูลหลักสูตร ข่าวสาร และบริการวิชาการของภาควิชาในรูปแบบที่ทันสมัยและเข้าถึงง่าย

🌐 Quick Links
Live Demo: https://fourgame.github.io/website_Health_Education_MU

Official Faculty Site: https://phhe.ph.mahidol.ac.th

✨ คุณสมบัติของเว็บไซต์ (Features)
ระบบเมนูนำทาง (Navigation): เชื่อมโยงไปยังหน้าหลัก, เกี่ยวกับเรา, บุคลากร, ข่าวสาร, บริการวิชาการ และการติดต่อ

ข้อมูลหลักสูตร: รายละเอียดหลักสูตรระดับบัณฑิตศึกษา ทั้งปริญญาโท (ภาคปกติและภาคพิเศษ) และปริญญาเอก (นานาชาติ)

ส่วนประชาสัมพันธ์: แสดงข่าวสารและประกาศล่าสุดของภาควิชา

เครือข่ายความร่วมมือ: แสดงโลโก้หน่วยงานภาคีที่ร่วมขับเคลื่อนด้านสุขศึกษา (เช่น สสส., ASH, HEPA)

Responsive Design: เว็บไซต์รองรับการแสดงผลบนหน้าจอหลายขนาด (Desktop & Mobile)

📁 โครงสร้างไฟล์หลัก (Project Structure)
index.html: หน้าแรกของเว็บไซต์ (Home)

about.html: ข้อมูลประวัติ วิสัยทัศน์ และทำเนียบผู้บริหาร

staff.html: รายชื่อและข้อมูลบุคลากร

news.html: ศูนย์รวมข่าวสารและประกาศ

Services.html: ข้อมูลบริการวิชาการ การฝึกอบรม และวารสารสุขศึกษา

contact.html: ช่องทางการติดต่อและแผนที่

/assets/: โฟลเดอร์เก็บไฟล์รูปภาพ (Logos, Models) และ CSS/JS (ถ้ามี)

🛠️ เทคโนโลยีที่ใช้ (Tech Stack)
HTML5 / CSS3: สำหรับโครงสร้างและการตกแต่งเว็บไซต์

GitHub Pages: สำหรับการโฮสต์เว็บไซต์แบบ Static

Icons & Images: สื่อประกอบจากคณะสาธารณสุขศาสตร์ มหาวิทยาลัยมหิดล
